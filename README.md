#baseMVC

It is simple basic MVC framework aimed to teach new developers how MVC works and teaches them to build one from scratch.

It has basic features similar to most MVC framework -

- Routing
- Model
- Controller
- View

#### Directory structure
Root
..|- bootstrap
....|- start.php
....|- autoloader.php
..|- assets
..|- vendor
..|- app
....|- Model
....|- View
....|- Controller
....|- Config
....|- Library
....|- init.php
....|- router.php
..|- index.php
